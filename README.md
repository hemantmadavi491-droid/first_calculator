# 🧮 Modern Flask Calculator

A sleek and responsive web-based calculator built with **Python Flask**, **HTML5**, and **CSS3**. Designed with a modern glassmorphism-inspired interface, this application delivers fast and accurate arithmetic calculations while providing a smooth and visually appealing user experience across desktop and mobile devices.

## ✨ Key Features

### 🎨 Modern UI Design
- Glassmorphism card layout
- Animated gradient background
- Smooth hover and transition effects
- Fully responsive design
- Clean and professional appearance

### 🔢 Calculator Functions
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Division-by-zero protection
- Input validation and error handling

### ⚡ Backend Functionality
- Built with Python Flask
- Handles GET and POST requests efficiently
- Secure form processing
- Dynamic result rendering using Jinja templates

## 🚀 How It Works

1. Enter two numbers.
2. Select an arithmetic operation.
3. Submit the form.
4. Flask processes the request.
5. The result is displayed instantly.
6. Errors are handled gracefully when invalid inputs are detected.
